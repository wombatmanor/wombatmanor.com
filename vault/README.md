# Obsidian vault

This folder will serve as the root of your Obsidian vault. You can open it in Obsidian and start creating notes.

## Suggested subfolders

- `daily/` – one markdown file per day (e.g., `2024-04-03.md`). Use a template for quick capture.
- `templates/` – reusable note templates (meeting notes, project briefs, etc.).
- `tags/` – optional: store tag‑specific notes or index files.

## Tagging convention (optional)

Use a forward‑slash hierarchy for tags, e.g. `#proj/xyz`, `#topic/philosophy`.
