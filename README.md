# WombatManor.com

Welcome to the internal repository for **WombatManor**.

This folder contains:
- `website/` – static website scaffold (HTML).  
- `docs/`   – company documents, policies, and reference material.  
- `projects/` – individual project directories.  
- `todos/` – task lists and planning notes.  
- `vault/` – Obsidian vault for knowledge‑base.

Use the provided `add_inbox.sh` script to capture ideas, then move them into appropriate locations.
